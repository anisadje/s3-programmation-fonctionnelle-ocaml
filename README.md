# Programmation fonctionnelle – OCaml (S3)

TP réalisés durant le semestre 3 en programmation fonctionnelle.

## Objectifs du module
- Comprendre les principes de la programmation fonctionnelle
- Manipuler les fonctions récursives
- Travailler avec les types algébriques
- Utiliser le pattern matching

## Contenu
- **TP1** : Fonctions récursives simples  
- **TP2** : Utilisation du module `Sys` et renforcement des notions de récursivité  
- **TP3** : Types algébriques et pattern matching  
- **TP4** : Structures récursives et utilisation des fonctions du module `List`  
- **TP5** : Manipulation de données structurées (CSV), filtres, statistiques et agrégations

## Ce que j’ai appris
- Récursion terminale
- Fonctions map, fold, filter
- Modélisation avec des types
- Programmation sans effets de bord
